# Social Network (SPA)

A Single Page Application (SPA) social network built with React, designed for a fast, smooth, and interactive user experience. Users can post, interact, customize profiles, manage friendships, and receive notifications.

## Features

- Create posts with text or images
- Like, comment, and share posts
- Upload images (posts, profile picture, and cover photo)
- Edit posts and comments
- Full profile customization
- Add and remove friends
- Block users
- Real-time notifications for likes, comments, friend requests, and more
- Search bar for finding users and posts

## Technologies

- React (SPA architecture)
- React Router
- Tailwind CSS for styling
- Supabase

## Project Highlights

- Fully interactive SPA with instant UI updates
- Friend system with add, delete, and block functionality
- Notification system to keep users updated in real time
- Responsive design for desktop and mobile
